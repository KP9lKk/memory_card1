### README: Memo Card — викторина на PyQt5

Программа представляет собой интерактивную викторину с выбором одного правильного ответа из четырёх вариантов. Реализована на Python с использованием библиотеки PyQt5.

#### Функционал

* отображение случайного вопроса из заранее подготовленного списка;
* выбор ответа с помощью радиокнопок (QRadioButton);
* проверка правильности ответа и визуальная обратная связь («Правильно!» / «Неверно!»);
* показ правильного ответа после попытки;
* подсчёт статистики: общее число вопросов и количество правильных ответов;
* расчёт рейтинга в процентах;
* переход к следующему вопросу по нажатию кнопки.

#### Интерфейс

Окно приложения включает:

1. **Текст вопроса** (QLabel) — в верхней части экрана.
2. **Группу вариантов ответа** (QGroupBox с QRadioButton) — четыре радиокнопки с вариантами ответа.
3. **Панель результата** (QGroupBox) — отображается после ответа:
   * статус («Правильно!» или «Неверно!»);
   * правильный ответ.
4. **Кнопку «Ответить» / «Следующий вопрос»** (QPushButton) — меняет текст в зависимости от этапа.

#### Использование

1. Запустите скрипт на Python (убедитесь, что установлен PyQt5: `pip install PyQt5`).
2. Прочитайте вопрос и выберите один из четырёх вариантов ответа.
3. Нажмите кнопку **«Ответить»**.
4. Ознакомьтесь с результатом и правильным ответом.
5. Нажмите кнопку **«Следующий вопрос»** для перехода к новому вопросу.
6. Продолжайте отвечать на вопросы — программа будет вести подсчёт правильных ответов и отображать текущий рейтинг.

#### Статистика

В консоли выводятся:
* общее количество заданных вопросов (`window.total`);
* количество правильных ответов (`window.score`);
* рейтинг в процентах (`(window.score / window.total) * 100`).

#### Структура данных

Вопросы хранятся в списке `questions_list` как объекты класса `Question`, который содержит:
* `question` — текст вопроса;
* `right_answer` — правильный ответ;
* `wrong1`, `wrong2`, `wrong3` — три неправильных варианта.

Пример добавления вопроса:
```python
questions_list.append(
    Question('Государственный язык Бразилии', 'Португальский', 'Английский', 'Испанский', 'Бразильский')
)
```

#### Технические детали

* **Язык:** Python 3.

* **Размер окна:** 400×300 px.
